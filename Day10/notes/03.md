### ‚úÖ 2. **Connect to MongoDB Atlas**

```python
from pymongo import MongoClient

# Replace <username>, <password>, and <cluster-url>
client = MongoClient("mongodb+srv://<username>:<password>@<cluster-url>/")

# Access a collection

```

## üîç PyMongo CRUD Operations

---

### üìå Insert Documents

#### ‚û§ Insert One:

```python
doc = { "id": 101, "name": "Alice", "department": "HR", "salary": 50000 }
collection.insert_one(doc)
```

#### ‚û§ Insert Many:

```python
docs = [
    { "id": 102, "name": "Bob", "department": "Sales", "salary": 60000 },
    { "id": 103, "name": "Charlie", "department": "IT", "salary": 70000 }
]
collection.insert_many(docs)
```

---

### üìå Find (Read) Documents

#### ‚û§ Find All:

```python
for doc in collection.find():
    print(doc)
```

#### ‚û§ Find with Filter:

```python
collection.find_one({ "id": 101 })
```

#### ‚û§ Find with Comparison:

```python
for doc in collection.find({ "salary": { "$gt": 60000 } }):
    print(doc)
```

#### ‚û§ Find Specific Fields (Projection):

```python
for doc in collection.find({}, { "name": 1, "salary": 1, "_id": 0 }):
    print(doc)
```

---

### üìå Update Documents

#### ‚û§ Update One:

```python
collection.update_one(
    { "id": 102 },
    { "$set": { "salary": 65000 } }
)
```

#### ‚û§ Update Many:

```python
collection.update_many(
    { "department": "IT" },
    { "$inc": { "salary": 5000 } }
)
```

---

### üìå Delete Documents

#### ‚û§ Delete One:

```python
collection.delete_one({ "id": 101 })
```

#### ‚û§ Delete Many:

```python
collection.delete_many({ "department": "Sales" })
```

---

### üìå Count Documents

```python
collection.count_documents({})  # All
collection.count_documents({ "department": "IT" })  # Filtered
```

---

### üìå Sort and Limit

```python
for doc in collection.find().sort("salary", -1).limit(2):
    print(doc)
```

---

### üìå Aggregation

#### ‚û§ Group by Department, Count:

```python
pipeline = [
    { "$group": { "_id": "$department", "count": { "$sum": 1 } } }
]
for result in collection.aggregate(pipeline):
    print(result)
```

#### ‚û§ Average Salary by Department:

```python
pipeline = [
    { "$group": { "_id": "$department", "avgSalary": { "$avg": "$salary" } } }
]
```

## ‚úÖ Mini Example Program

```python

    print(emp)
```
